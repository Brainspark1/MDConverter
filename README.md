# MDConverter

MDConverter is a lightweight, native macOS application that allows you to quickly convert between Markdown and Rich Text. The app is designed to be simple, fast, and distraction-free, making it easy to switch between Markdown syntax and formatted text.

## Features

- Convert Markdown to Rich Text
- Convert Rich Text back to Markdown
- Lightweight and fast native macOS application
- Clean and simple interface
- Useful for documentation, notes, and writing

## Installation

You can install MDConverter by downloading the DMG file from the official website.

Website:
https://md-converter-website.vercel.app

Steps:

1. Visit the website.
2. Download the MDConverter.dmg file.
3. Open the downloaded DMG file.
4. Drag the MDConverter app into your Applications folder.
5. Launch MDConverter from Applications.

## How to Use

1. Paste or type Markdown text into the editor.
2. Convert it to Rich Text instantly.
3. You can also paste Rich Text and convert it back to Markdown.

This allows you to easily move between formatted text and Markdown syntax.

## System Requirements

- macOS 13 or later
- Intel or Apple Silicon Mac
